Getting Started¶

A multi-agent development system where you describe what you want to build in plain language, and eight specialized AI agents plan, develop, test, secure, and ship it — orchestrated in real time throu

This guide covers the prerequisites, installation, and project structure for agent-orchestra.

Prerequisites¶

  • TypeScript development environment
  • Python development environment
  • Markdown development environment
  • JSON development environment
  • Shell development environment
  • YAML development environment
  • Dockerfile development environment
  • JavaScript development environment

Installation¶

# Clone the repository
git clone https://github.com/morbidsteve/agent-orchestra
cd agent-orchestra

npm install

yarn install

pnpm install
