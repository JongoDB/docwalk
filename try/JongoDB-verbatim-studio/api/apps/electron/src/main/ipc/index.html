ipc.ts¶

Registers IPC handlers for communication between the main process and renderer, exposing functions like app:getVersion, api:getUrl, and dialog:openDirectory. Handles requests from the renderer process, providing access to backend APIs and system dialogs. Consumed by the renderer process to interact with the main process and backend services.
