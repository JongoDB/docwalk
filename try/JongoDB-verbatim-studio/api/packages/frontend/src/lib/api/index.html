api.ts¶

Orchestrates API interactions by resolving the backend API URL and managing auth tokens. Exposes onAuthRequired(), setAuthTokens(), clearAuthTokens(), and hasAuthToken() for authentication state management. Consumed by the frontend application to handle API requests and authentication. Connects to the backend API via HTTP requests.
