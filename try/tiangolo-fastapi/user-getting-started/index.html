Getting Started¶

Getting Started with FastAPI¶

Welcome to FastAPI, a modern, fast (high-performance), web framework for building APIs with Python 3.7+ based on standard Python type hints. In this tutorial, we'll guide you through the installation process and a practical "first use" walkthrough.

Step 1: Installation¶

To get started with FastAPI, you'll need to have Python 3.7+ installed on your machine. You can check your Python version by running: 

python --version
If you don't have Python 3.7+, you can download it from the official Python website.

Next, you'll need to install FastAPI and an ASGI server, such as Uvicorn. We recommend using pip, the Python package manager. Run the following command: 

pip install fastapi uvicorn
This command installs FastAPI and Uvicorn.

Step 2: Create Your First FastAPI Application¶

Create a new file called main.py and add the following code: 

from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"Hello": "World"}
Let's go through what this code does:

  • We import the FastAPI class from the fastapi module.
  • We create a new instance of the FastAPI class, which is our application.
  • We define a route for the root URL ("/") using the @app.get() decorator.
  • The read_root function returns a JSON response with the message {"Hello": "World"}.

Step 3: Run Your FastAPI Application¶

To run your FastAPI application, navigate to the directory where you created the main.py file and run: 

uvicorn main:app --reload
Here's what this command does:

  • uvicorn is the ASGI server we're using to run our application.
  • main is the name of the Python file where our application is defined.
  • app is the name of the FastAPI application instance.
  • --reload enables auto-reload, which means the server will restart automatically when you make changes to your code.

You should see output similar to: 

INFO: Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)

Step 4: Test Your FastAPI Application¶

Open a web browser and navigate to http://127.0.0.1:8000/. You should see the JSON response: 

{
  "Hello": "World"
}
Congratulations! You've just created and run your first FastAPI application.

What's Next?¶

  • Explore the FastAPI documentation to learn more about its features and capabilities.
  • Try adding more routes and endpoints to your application.
  • Experiment with different data types and validation using Python type hints.

We hope this tutorial has helped you get started with FastAPI. Happy building!
