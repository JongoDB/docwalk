database.ts¶

Orchestrates type safety for database interactions by defining the full PostgreSQL schema as TypeScript types. Exposes the Json utility type, the top‑level Database type, and nested Row, Insert, and Update types for households, household_invites, household_members, and storage_units. Consumed by data‑access, service, and API layers to guarantee compile‑time validation of queries and payloads. Acts as the canonical contract between the application code and the underlying database schema.
