price-utils.ts¶

Orchestrates price normalization and best‑price selection by converting units, calculating per‑unit costs, and aggregating price history. Exposes interfaces PriceHistoryItem and BestPrice, and functions convertToBaseUnit, calculatePricePerUnit, calculateBestPrices, formatPrice, and formatPricePerUnit for downstream pricing and recommendation services. Consumed by the pricing API layer to present normalized pricing to UI components.
