Getting Started¶

A Docker Compose deployment showcasing the Sliver C2 framework for cybersecurity training and demonstration purposes. This environment includes a weather website as a cover application, nginx as a reverse proxy, and Sliver C2 server for command and control operations.

This guide covers the prerequisites, installation, and project structure for sliver-weather.

Prerequisites¶

  • JavaScript development environment
  • Dockerfile development environment
  • YAML development environment
  • Shell development environment
  • JSON development environment
  • Markdown development environment

Installation¶

# Clone the repository
git clone https://github.com/JongoDB/sliver-weather
cd sliver-weather

npm install

yarn install

pnpm install
