Getting Started¶

Getting Started with sliver‑weather¶

Welcome to sliver‑weather!
This quick guide will walk you through installing the tool, running your first command, and getting the most out of it – all in just a few minutes.

Tip: If you run into any hiccups, feel free to ask for help on the project’s GitHub issues page or drop a comment in the community chat.

1. Prerequisites¶

Item Minimum Version Why you need it
Git 2.20+ To clone the repository
Python 3.8+ sliver‑weather is a Python CLI tool
pip 20+ Python’s package installer

Windows users: You can install Git from git-scm.com and Python from python.org. Make sure to add both to your system’s PATH.

2. Clone the Repository¶

Open a terminal (or Command Prompt / PowerShell on Windows) and run:

git clone https://github.com/your-org/sliver-weather.git
cd sliver-weather

🎉 You’ve just downloaded the source code! The next step is to set up a virtual environment so that the tool’s dependencies don’t interfere with other Python projects on your machine.

3. Create a Virtual Environment¶

# Create a new virtual environment named "venv"
python -m venv venv

# Activate it
# • On macOS/Linux:
source venv/bin/activate
# • On Windows (cmd.exe):
venv\Scripts\activate.bat
# • On Windows (PowerShell):
venv\Scripts\Activate.ps1

You’ll know it’s active because your prompt will now start with (venv).

4. Install the Dependencies¶

pip install --upgrade pip
pip install -e .

The -e flag installs the package in “editable” mode, so any changes you make to the code will be reflected immediately.

5. Verify the Installation¶

Run the built‑in help command to see all available options:

sliver-weather --help

You should see something like:

Usage: sliver-weather [OPTIONS] COMMAND [ARGS]...

  A lightweight CLI for fetching weather information.

Options:
  --version  Show the current version and exit.
  --help     Show this message and exit.

Commands:
  get   Get the current weather for a location.
  forecast   Get a short‑term forecast.

If you see this, you’re ready to start using sliver‑weather!

6. First Use: Get the Current Weather¶

Let’s fetch the weather for your city. Replace YOUR_CITY with a real city name (e.g., London).

sliver-weather get "YOUR_CITY"

Example:

sliver-weather get "San Francisco"

You should see output similar to:

🌤️  San Francisco, US
Temperature: 18°C
Condition: Partly Cloudy
Wind: 5 km/h NE

7. Quick‑Start: Forecast¶

Want to see a short‑term forecast? Try:

sliver-weather forecast "YOUR_CITY" --days 3

Example:

sliver-weather forecast "Tokyo" --days 3

Output will list the weather for the next three days.

8. Customizing the Output¶

sliver‑weather supports a few handy flags:

Flag Description
--units metric Show temperatures in Celsius (default).
--units imperial Show temperatures in Fahrenheit.
--format json Output data as JSON for scripting.

Example:

sliver-weather get "Berlin" --units imperial --format json

9. Uninstalling¶

If you ever want to remove the tool:

deactivate   # exit the virtual environment
pip uninstall sliver-weather

10. Troubleshooting¶

Issue Fix
sliver-weather: command not found Make sure the virtual environment is activated and that pip install -e . ran successfully.
API errors The tool uses the OpenWeatherMap API. Ensure you have a valid API key set in the environment variable OWM_API_KEY.
“Permission denied” On macOS/Linux, you may need to run chmod +x sliver-weather to make the script executable.

11. Want to Contribute?¶

Great! Check out the CONTRIBUTING.md file in the repo for guidelines on how to submit pull requests, report bugs, or add new features.

🎉 You’re All Set!¶

You now have a fully functional weather CLI on your machine. Explore the commands, experiment with different locations, and enjoy the sunshine (or rain) data at your fingertips.

Happy coding! 🌞
