Getting Started¶

FastAPI framework, high performance, easy to learn, fast to code, ready for production

This guide covers the prerequisites, installation, and project structure for fastapi.

Prerequisites¶

  • Markdown development environment
  • Python development environment
  • YAML development environment
  • Shell development environment
  • JavaScript development environment
  • TOML development environment

Installation¶

# Clone the repository
git clone https://github.com/fastapi/fastapi
cd fastapi

# Install dependencies
poetry install
