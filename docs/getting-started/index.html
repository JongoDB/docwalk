Getting Started¶

Your codebase, documented. Automatically. Analyze any repo, generate MkDocs Material sites, deploy anywhere, and keep docs in sync with git-diff intelligence.

This guide covers the prerequisites, installation, and project structure for docwalk.

Prerequisites¶

  • TypeScript development environment

Installation¶

# Clone the repository
git clone <repository-url>
cd docwalk

npm install

yarn install

pnpm install
