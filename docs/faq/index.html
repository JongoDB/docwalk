Frequently Asked Questions¶

What is docwalk?

Your codebase, documented. Automatically. Analyze any repo, generate MkDocs Material sites, deploy anywhere, and keep docs in sync with git-diff intelligence.

What languages/technologies does docwalk use?

typescript (100%).

What commands are available?

Available commands: docwalk ci-setup, docwalk deploy, docwalk dev, docwalk doctor, docwalk generate, docwalk init, docwalk status, docwalk sync, docwalk undeploy, docwalk version-list, docwalk version-deploy. See the Getting Started guide for usage details.
