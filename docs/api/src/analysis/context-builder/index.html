Smart Context Builder

Smart Context Builder Assembles ranked code chunks from the dependency graph for LLM calls. Provides most of RAG's benefits without embedding/vector dependencies. Relevance scoring uses the existing dependency graph: - Direct dependency: 1.0 - 1-hop transitive: 0.6 - Same directory: 0.4 - Same logical section: 0.3 - README/docs: 0.2
